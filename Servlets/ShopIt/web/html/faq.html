Frequently Asked Questions

What is this website for?

This website helps people decide what products they could buy based on other users' reviews.

How to use this website?

Login to the Home page and type in the product you want to search about. This will automatically give a list of reviews on that product.

Is there a limit to the number of reviews for a particular user?

No. One user can write any number of reviews for any number of products. In fact, they can write 2 reviews for the same product too.
